## Aplikasi Inventa'ta

Aplikasi **Inventa'ta** merupakan aplikasi pencatatan inventori yang ditujukan khusus untuk **pedagang sandal**.  
Aplikasi ini berguna untuk mempermudah pencatatan serta memantau stok sandal di gudang secara efisien.

### Deskripsi
Proyek ini dibuat untuk memenuhi tugas skripsi sekaligus sebagai hasil penelitian dengan judul:

**â€œRancang Bangun Sistem Inventory Berbasis Android Menggunakan SQLite pada Pedagang Sandal di Pasar Toddopuliâ€**

Aplikasi ini dirancang sebagai solusi untuk mengatasi berbagai permasalahan, seperti:
- Bias data terhadap barang yang sudah habis
- Kesulitan memantau stok barang di gudang

Dengan beberapa pertimbangan, aplikasi dibuat:
- Dapat digunakan **secara offline**
- Tidak memerlukan biaya tambahan untuk database maupun hosting

### Fitur Utama
- Pencatatan barang masuk dan barang keluar
- Notifikasi stok hampir habis dan stok habis
- Pencarian data barang dengan mudah

### Pertimbangan Pembuatan Aplikasi
- Biaya pajak pasar yang semakin tinggi
- Suhu pasar yang panas
- Catatan barang sering hilang atau rusak akibat banjir
- Kesulitan melakukan restok karena tidak mengetahui jumlah barang di gudang

### Solusi yang Diberikan Aplikasi
- Menggunakan database **SQLite**, sehingga tidak memerlukan biaya penyimpanan tambahan
- Aplikasi berjalan **secara offline** dan dapat digunakan kapan saja
- Menghemat baterai perangkat karena tidak membutuhkan koneksi internet
- Digitalisasi pencatatan mencegah kehilangan atau kerusakan data
